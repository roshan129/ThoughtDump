# Sprint 1 — Project Setup + UI Skeleton

## Goal

Basic working UI with navigation (no AI yet)

## Stories

### 1.1 Project Initialization
- Setup React app (Next.js)
- Install Tailwind CSS
- Setup folder structure:
  - `components/`
  - `pages/`
  - `services/`

### 1.2 Layout System
- Create reusable layout:
  - Sidebar
  - Topbar

### 1.3 Sidebar Component
- Items:
  - Dashboard
  - My Thoughts
  - Content Library
  - Settings

### 1.4 Dashboard UI
- Large textarea
  - Placeholder: "What’s on your mind?"
- Dropdowns:
  - Tone selector
  - Format selector
- Button:
  - "Generate Content"

### 1.5 Routing
- Setup routes:
  - `/dashboard`
  - `/outputs`
  - `/thoughts`

## Acceptance Criteria
- App runs locally
- Navigation works
- UI is clean and usable
