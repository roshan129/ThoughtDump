# Sprint 3 — Output UX (Make It Addictive)

## Goal

Polished output experience

## Stories

### 3.1 Output Cards
Each card includes:
- Content text
- Copy button
- Regenerate button

### 3.2 Copy Feature
- Copy to clipboard
- Show toast: "Copied!"

### 3.3 Multiple Variants
- Label outputs:
  - Viral
  - Deep
  - Thread

### 3.4 Regenerate Feature
- Regenerate single card
- Regenerate all

### 3.5 Smooth UX
- Animations
- Loading shimmer

## Acceptance Criteria
- Outputs look clean
- Copy works perfectly
- UX feels fast
